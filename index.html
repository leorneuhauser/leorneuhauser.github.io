Talks
  • Rigid algebras are right adjoint to cospans - YTM, Münster 2024 (slides)

    A talk about an upcoming paper on rigid algebras and cospans. I define the notion of a rigid algebras and indicate some proofs, including the adjunction with cospans.

Seminar notes
  • Hirzebuch's signature theorem - Jerusalem 2024

    This talk was part of a seminar on orientations. In this talk we define the signature of a manifold, prove that it is a genus, and prove that it equal to the L-polynomial in Pontrjagin classes.

  • Cyclotomic redshift - Oberwolfach 2024

    This talk was prepared for the Oberwolfach Arbeitsgemeinschaft about the disproof of the telescope conjecture. In this talk we proved cyclotomic redshift, namely that T(n+1)-localized K-theory sends height n cyclotomic extensions to height n+1 cyclotomic extension.

  • Proving the chromatic nullstelenzats - Jerusalem 2024

    This talk was prepared for the Kazhdan seminar on the chromatic nullstelenzats. In this talk we gathered all the results proven throughout the semester to finalize the proof of the chromatic nullstelenzats.

  • Cyclotomic and polygonic spectra - Jerusalem 2024

    This talk was prepared for a seminar on the disproof of the telescope conjectrure. In this talk we define cyclotomic and polygonic spectra, and using them we define THH, TC and TR. We also prove some adjunctions.
